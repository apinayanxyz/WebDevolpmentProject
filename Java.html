Regex

Understanding how regex function can prove to be difficult, so here are some basic examples

  • Regex for a full name
    ([A-Z]{1}[a-z]+)( )([A-Z]{1}[a-z]+)
  • Regex pattern for an email address(.com)
    ([A-za-z0-9]+[@])([A-Z]{1}[a-z]+[][.com])
  • Regex pattern for an email address(.co.uk)
    ([a-z]+[@])([A-Z]{1}[a-z]+[].co][.uk])
  • Regex pattern for a word with no numbers or extra characters
    ([a-z])
  • Regex pattern for a capital word with no numbers or extra characters
    ([A-Z]{1}[a-z]+)

The website provided below should allow you to test out your patterns and see if they work as intended

https://regex101.com/
